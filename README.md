# Проект программы коррекции EPROM батарейного модуля APC SYBT5

## Описание
Программа коррекции EPROM батарейного модуля APC SYBT5 позволяет:
- Загрузить исходный файл EPROM в формате .bin
- Изменить дату замены аккумуляторных батарей
- Изменить серийный номер батарейного модуля APC SYBT5
- Сохранитть итоговый файл EPROM в формате .bin

## Стек
Python 3.9 — среда выполнения и язык разработки проекта.
